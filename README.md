# Início

- Inicializar Repositório

```npm
npm init -y
```

- Instalar Pacote: Knex

```npm
npm install knex --save
```

- Instalar Pacote: Biblioteca Base do Banco

mysql2 é mais recente e performa melhor

```npm
npm install mysql2 --save
```
